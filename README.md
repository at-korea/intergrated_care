# 전국 통합돌봄서비스 대시보드 · Integrated Care Korea

대한민국 17개 시·도 226개 시·군·구의 보건복지부 **지역사회 통합돌봄** 사업 12,104건을 검색·비교·분석할 수 있는 인터랙티브 대시보드입니다.

🌐 **Live Demo**: https://at-korea.github.io/intergrated_care/

## 주요 기능

| 기능 | 설명 |
|---|---|
| 🗺 **지도 뷰** | 시·도 / 시·군·구 두 단계 줌. 색상 모드: 총 사업 수 / 시·군·구 평균 / 인구 1만명당 / 지역특화 비율 |
| 🔍 **검색** | 시·도 → 시·군·구 cascading 풀다운, 카테고리·사업구분·키워드 조합 필터 |
| 📊 **시·군·구 프로필** | 선택한 지역의 인포그래픽, 전국 평균 대비 강·약 영역, 인구 보정 위치 |
| 🔄 **비교 모드** | 2~4개 시·군·구 동시 비교, 사업 매트릭스(공통/특이 색상 구분) |
| 👥 **인구 보정** | 228개 시·군·구 인구 결합, 1만명당 사업 수 계산 |
| 📄 **PDF 리포트** | 시·군·구별 메뉴판을 한 번 클릭으로 PDF 생성(자동 페이지 분할) |
| 📑 **엑셀 다운로드** | 현재 필터 결과를 xlsx로 |
| 🔗 **URL 공유** | 현재 탭·필터 상태가 URL에 인코딩 |

## 데이터

`data/` 폴더에 raw 데이터가 포함되어 있습니다.

- `services_national.json` (4.4 MB) — 12,104개 사업 레코드 (시도, 시군구, 대분류, 중분류, 사업명, 사업내용, 지원대상, 제공기관, 사업구분)
- `stats.json` (168 KB) — 시·도/시·군·구별 사전 계산 통계 (대분류 분포, 인구, 1만명당 등)
- `population.json` (8 KB) — 228개 시·군·구 인구 (행정안전부 주민등록 2024)
- `korea_sgg.geojson` (901 KB) — 250개 시·군·구 단순화 GeoJSON (KOSTAT 2018)
- `korea_sido.geojson` (343 KB) — 17개 시·도 단순화 GeoJSON

## 출처

- **사업 데이터**: 보건복지부 [지역사회 통합돌봄](https://www.mohw.go.kr/menu.es?mid=a60600000000) 전용 누리집의 시·군·구별 서비스 메뉴판 PDF (2026.03~04 기준) 자동 파싱
- **인구 데이터**: 행정안전부 주민등록 인구통계 (2024)
- **지도 데이터**: 통계청 KOSTAT GeoJSON (2018) — [southkorea/southkorea-maps](https://github.com/southkorea/southkorea-maps)

## 기술

- 순수 HTML/CSS/JavaScript — 백엔드 없이 단일 HTML 파일로 동작
- 모든 데이터 클라이언트 임베드 (오프라인 동작 가능, 라이브러리만 CDN)
- **CDN 라이브러리**: Chart.js, jsPDF, html2canvas, SheetJS
- **폰트**: Noto Sans KR (Google Fonts)

## 데이터 품질·한계

- 234개 PDF의 표 구조가 시·도마다 달라 컬럼 매핑/셀 병합 보정 후 추출. 일부 셀은 줄바꿈 분리 오류 가능성
- 사업구분이 "미분류"인 사업이 약 3,800건 — 충북·충남·전북·전남 일부 PDF에서 사업구분 컬럼 누락
- 지원대상은 자유기술 형식이라 9개 표준 그룹으로 정규화
- 본 분석은 절대 건수 기준이며, 사업 메뉴 수는 실제 수혜자/예산 규모와 다를 수 있음
- 일반구가 있는 시(수원·성남·안양·고양·용인·안산·청주·천안·전주·포항·창원)는 시 단위로 통합 매핑

## 라이선스

데이터 출처는 보건복지부·행정안전부·통계청 공공데이터입니다. 본 시각화 코드는 자유롭게 참고·수정·재배포할 수 있으나, 정책 연구·기획 참고용으로 제작된 **비공식 시각화**입니다. 정확한 사업 신청·이용은 해당 시·군·구 보건소 또는 복지부서로 문의해 주세요.

## 로컬 실행

```bash
git clone https://github.com/at-korea/intergrated_care.git
cd intergrated_care
# 단일 HTML이므로 그냥 브라우저에서 dashboard.html 열기
# 또는 로컬 서버 실행:
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000
```

## 만든 방법

1. 보건복지부 사이트에서 234개 시·군·구 PDF 메뉴판 수집
2. `pdfplumber`로 표 구조 자동 파싱 + 시·도별 컬럼 매핑 보정
3. 데이터 정규화 (대분류 통일, 사업구분 정리, 시·군·구 매핑)
4. 시·도/시·군·구 통계 사전 계산, 한국 GeoJSON 단순화
5. 단일 HTML에 데이터 임베드 + 인터랙티브 UI 작성
